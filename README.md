# MyFlix: Group Members: Altynay Bolatova, Farida Dovletbaeva, Sagyndyk Alua

## Project Overview

The **MyFlix** project is a dynamic, multi-page, and responsive web application focused on **providing users with an intuitive platform to explore, search, rate, and save information on the latest and classic movies and TV series.** The application features a complete user authentication system and leverages an external API to deliver rich, dynamic content, creating a personalized viewing guide for every user.
